### 디렉토리와 파일

> CLI (Command Line Interface)
> 
- `ls`, `ls -al` : 현재 디렉토리에 존재하는 파일 확인
- `pwd` : 현재 디렉토리 경로
- `mkdir` : 폴더 생성
- `ls -l` : 현재 디렉토리의 파일 자세한 정보 (list)
    - - : 파일
    - d : 디렉토리
- `rm` : 파일 삭제

### —help와 man

- 사용설명서 **확인**
    - `<CIL> —help`
        - mac 터미널 환경에서 X
    - `man <CIL>`
        - 창 안에서 `/검색어` 로 검색
